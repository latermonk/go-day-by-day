https://golang.org/dl/   

##  安装

解压并放到 /usr/local/目录下


##  配置环境变量


###   配置go环境变量 

export PATH="$PATH:/usr/local/go/bin"


export GOPATH="$PATH:/home/www/importgp"


source the configuration file
